Advanced Image Restoration

A deep generative model specialized in transforming grayscale archives into vibrant, realistic imagery using adversarial training.

Project Overview

Implemented a Generative Adversarial Network (GAN) using the Pix2Pix framework to automate photo colorization. The system utilizes a U-Net generator with skip connections to maintain spatial integrity and realistic texture reproduction.

Deep Learning Highlights: CIE Lab color space optimization (predicting AB channels from L), Adversarial + L1 multi-component loss, and automated preprocessing pipelines for large-scale image datasets.
