# Portfolio Demos

The portfolio website features interactive demonstrations of capabilities. These are not static images but functioning code designed to illustrate specific technical concepts.

## 1. AlpineStay Luxury
A premium booking platform for high-end cabins.
*   **Purpose:** To demonstrate "quiet luxury" in web design.
*   **Key Design Elements:** Cinematic scroll interactions, high-fidelity imagery, immersive atmosphere, and a frictionless booking calendar.
*   **Technical:** Responsive layout, Vanillas JS animations, custom availability logic.

## 2. THE MR Studio
A comprehensive management system for a boutique salon.
*   **Purpose:** To show a full-stack business solution.
*   **Key Design Elements:** Clean, professional interface for both clients and administrators.
*   **Technical:** React-based frontend, robust booking logic, reduction of administrative overhead (specifically no-shows).

## 3. Interactive Automation Demo (Nina's Assistant)
An embedded AI agent directly in the portfolio.
*   **Purpose:** To demonstrate real-time AI capabilities and contextual awareness.
*   **Function:** It answers questions about the portfolio itself, acting as a first layer of contact.
*   **Technical:** Orchestrated via n8n, connecting user input to an LLM with specific instructions about Nina's services.
