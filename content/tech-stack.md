# Technology Stack

The tools chosen are robust, widely supported, and capable of scaling. The selection prioritizes stability and flexibility over temporary trends.

## Core Development
*   **HTML5 & CSS3:** Semantic structure and modern styling.
*   **JavaScript (ES6+):** The logic layer for all interactive web applications.
*   **React:** A library for building dynamic, component-based user interfaces.
*   **Node.js:** Server-side execution for backend logic and API handling.

## Automation & AI
*   **n8n:** A workflow automation tool used to orchestrate complex data flows between applications. It allows for high customizability and privacy.
*   **OpenAI API (GPT-4o):** The intelligence layer for text processing, reasoning, and content generation.
*   **LangChain:** A framework for developing applications powered by language models, used for more complex agentic behaviors.
*   **Pinecone / Vector Databases:** Storing and retrieving knowledge for AI context (RAG systems).

## Infrastructure & Tools
*   **Git & GitHub:** Version control and code collaboration.
*   **Netlify / Vercel:** Modern hosting platforms for fast, secure web deployment.
*   **Tailwind CSS:** A utility-first CSS framework for rapid UI development (used when appropriate for the project scale).
